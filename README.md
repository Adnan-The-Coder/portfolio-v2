## Getting Started

First, run the development server:

```bash
bun run dev
```
If bun is not installed 
<br>
For Windows
```bash
powershell -c "irm bun.sh/install.ps1 | iex"
```
For Mac
```bash
npm install bun -g
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
